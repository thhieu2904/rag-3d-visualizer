# Tài liệu tham khảo (References)

Thư mục này chứa các tài liệu tham khảo cho việc dựng hình và demo 3D.

## Các file đã được đẩy lên (Tracked)

- **`Đại học Đà Nẵng.html`**: File HTML chứa nội dung tham khảo chính (Visualize structure).

## Các file bị loại bỏ (Ignored)

Các file/thư mục sau không được đẩy lên GitHub do kích thước quá lớn hoặc chứa nhiều file nhỏ gây lỗi khi push:

- **`Đại học Đà Nẵng_files/`**: Thư mục resource đi kèm với file HTML (chứa các assets, scripts, images chi tiết của trang web tham khảo).
- **`dynamic_state1761293797730.png`**: Ảnh chụp trạng thái động (kích thước lớn ~30MB).

> **Lưu ý:** Nếu cần xem đầy đủ nội dung tham khảo, vui lòng kiểm tra thư mục này trên máy local (nơi chứa đầy đủ assets). Trên repo chỉ lưu trữ file HTML cấu trúc chính.
